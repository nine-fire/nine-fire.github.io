文本标签及样式

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础

<em> 和 <strong>

  • em 标签用于表示一段内容中的着重点。主要表示语气上的强调。
    strong 标签用于表示一个内容的重要性。比 em更加强烈。
    这两个标签可以单独使用，也可以一起使用。

例：

  • 通常 em 显示为斜体，而 strong 显示为粗体。

<i> 和 <b>

  • i 标签中的内容会以斜体显示
  • b 标签中的内容会以粗体显示

但是 i 标签和 b 标签没有任何语义，在HTML5核心规范中，对于不需要强调的内容而是单纯的斜体或者粗体，则使用 i 标签和 b 标签。

<small>

  • small 标签中的内容会比他的父元素中的文字小一些
    在HTML5中使用 small 标签来表示一些细则类的内容。
    比如：合同中的小字、网页中的版权声明都可以放在 small 标签中。

当然也有 big 标签，但是它没有任何语义，已经被 HTML 5 淘汰，我们不使用。

<cite>

  • 使用 <cite> 标签可以指明对某内容的引用或参考。
    页面中所有加书名号的内容都可以写在 cite 标签中。
    比如，戏剧、文章或图书的标题，歌曲、电影、照片或雕塑的名称等。

例：

<q> 和 <blockquote>

  • q 表示一个短引用（行内应用）。
    浏览器会默认加上双引号。

例：

  • blockquote 表示一个长引用（块级引用）。
    注意：blockquote 是一个块级元素，所以不能放在 p 标签中。

<sup> 和 <sub>

  • 使用 sup 标签来设置一个上标。
    例：X<sup>2</sup> /*表示 X 的平方*/
  • 使用 sub 标签来设置一个下标。
    例：H<sub>2</sub>O /*表示化学中水的符号*/

<ins> 和 <del>

  • ins 表示插入的内容，显示时通常会加上下划线。
    就像填空题的感觉。
  • del 表示删除的内容，显示时通常会加上删除线。

例：

<code> 和 <pre>

  • 如果你的内容包含代码示例或文件名，就可以使用 code 元素，但是 code 是语义化标签，不会保留格式。
  • pre 元素表示的是预格式化文本，不会忽略多个空格，可以使用 pre 包住 code 来表示一段代码。
