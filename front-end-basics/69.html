CSS Hack

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础

在一些特殊情况，需要某一段代码仅在某些特殊浏览器（IE，不具体点名）中执行，而在其他浏览器中忽略，此时则可以使用CSS Hack。

条件Hack

条件Hack仅对IE浏览器有效，其他浏览器都会将其识别为注释。
仅支持IE10以下的浏览器，不包括IE10.

语法：

keywords：
空：是否
gt：大于（greater than）
gte：大于等于（greater than or equal）
lt：小于（less than）
lte：小于等于（less than or equal）
!：非指定版本

version：6、7、8、9

例：

属性级Hack

语法：选择器{<hack>?样式名:样式值<hack>?;}
在样式前添加：
_： 选择IE6及以下
*：选择IE7及以下

在样式后添加：
\9：选择IE6以上
\0：选择IE8+和Opera15以下

Hack有风险，使用需谨慎。
CSS Hack不到万不得已时尽量不要使用。

条件符级别Hack

* html 选择器{样式名:样式值}：仅选择IE6
