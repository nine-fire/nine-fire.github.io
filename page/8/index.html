盒子模型基本概念

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础
  • CSS处理网页时，它认为每个元素都包含在一个不可见的盒子里。
    如果把所有的元素都想象成盒子，那么我们对网页的布局就相当于在摆放盒子。
    我们只需要将相应的盒子摆放到网页中相应的位置即可完成网页的布局。
  • 一个盒子我们会分成以下几个部分：
    content：内容区
    padding：内边距
    border：边框
    margin：外边距

    此部分只可意会不可言传。

其他文本样式

  • text-transform样式用于设置英文字母的大小写
    可选值：
    • none：有小写和大写字母的标准文本（默认值）

    • lowercase：只有小写字母

    可选值：
    • none：不添加任何装饰（默认值）
    • underline：添加下划线
    • overline：添加上划线
    • line-through：添加删除线

行间距

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础

在CSS中并没有提供一个直接设置行间距的方式，我们只能通过设置行高来间接地设置行间距，行高越大则行间距越大。

  • 使用line-height来设置行高
    行高类似于我们日常生活中使用的练习本，练习本是由若干条水平的直线绘制而成，那么线与线之间的距离就是行高。
    网页中的文字实际上也是写在一条条看不见的水平线之间的，而文字默认会在行高中垂直居中显示。

字体的其他样式

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础
  • 使用font-style来设置文字是否倾斜
    可选值：
    • normal，文字正常显示（默认值）
    • italic，文字会以斜体显示
    • oblique，文字会以倾斜的效果显示

italic是使用系统中已安装的斜体字体来显示斜体，而oblique是强制扭曲字体来显示斜体。
大多数浏览器都不会对斜体和倾斜作区分，设置italic或oblique的显示效果往往是相同的，因此开发中一般只会使用italic。

字体的分类

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础
  • serif（衬线字体）
    衬线字体，在字的笔画开始、结束的地方有额外的装饰，而且笔画的粗细会有所不同。
    例如宋体就属于衬线字体。
    它强调了每个字母笔画的开始和结束（强调于一个单词），因此易读性比较高，在整文阅读的情况下，适合使用serif字体进行排版，易于换行阅读的识别性，避免发生行间的阅读错误。

  • 使用color来设置文字的颜色
    例：color: red; /*设置文字颜色为红色*/
  • 使用font-size来设置文字的大小
    例：font-size: 22px;/*设置文字大小为22像素*/

font-size设置的并不是文字本身的大小，在页面中，每个文字实际上都是处在一个看不见的“框”中，font-size实际上设置的是“框”的高度，并不是文字的大小。一般情况下文字都要比这个“框”小一些，有时也会比“框”大，由于字体的不同，显示效果也不尽相同。

颜色的表示

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础
  • 在CSS中可以直接使用颜色的英文单词来表示不同的颜色。
  • 也可以使用RGB值来表示不同的颜色。
    所谓RGB值指的是通过red、green、blue三原色按照不同浓度进行混合，来表示出千千万万的颜色。
    语法：rgb(红色的浓度,绿色的浓度,蓝色的浓度);
    颜色的浓度是一个0255的十进制数值，255表示最浓，0表示没有。
    颜色的浓度也可以是一个百分数，0%    100%

长度单位

发表于 2018-06-06 更新于 2018-08-16 分类于 学习笔记 ， HTML+CSS基础

像素：px

  • 像素是我们在网页中最常用的一个单位。
    一个像素就是我们屏幕中的一个小点，我们的屏幕实际上是由一个个足够小的像素点构成的，一般不能用肉眼直接看见。
  • 不同显示器中一个像素的物理尺寸也不尽相同。
    一般地，像素的物理尺寸越小，显示效果越好，反之显示效果相对差。

问题：手机屏幕像素的物理尺寸大约是电脑屏幕像素的25%，我们在开发过程中要特别注意。但手机浏览器一般会自动将你设置的像素值乘以4再进行渲染。
